This folder contains the solution to the proposed problem.
Notebooks are annotated and should be clear to follow. All material is already printed.
If you would like to re-run the notebooks, please make sure you have all the needed libraries up to date in Python3. The imports are all called on the top cell of the notebook.

The files included are:

- Initial_study.ipynb: 
This notebook contains the initial analysis

- Learning_Notebook.ipynb:
This notebook contains further analysis and constructions of the ML models.

- contracts.csv / transactions.csv
Initial datasets 

- contracts_updated.csv / transactions_updated.csv
Datasets enriched with the notebook "Initial_study.ipynb"

- model_1_plot.png / model_2_plot.png:
Printed graphs of the two architectures used in "Learning_Notebook.ipynb"
